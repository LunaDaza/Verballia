# Verbalia: El Reino de las Palabras Perdidas

Verbalia es un RPG educativo en 3D desarrollado con HTML, CSS, JavaScript y Three.js. El juego combina exploración, narrativa y desafíos de lenguaje dentro de un mundo medieval fantástico donde la magia nace de las palabras.

El jugador asume el papel de un héroe destinado a restaurar el conocimiento perdido del reino después de que Morgalix, la Bruja del Silencio, lanzara una maldición que hizo que los habitantes olvidaran cómo escribir correctamente. A través de misiones, acertijos e interacciones con personajes, el jugador deberá recuperar fragmentos de conocimiento y devolver la magia a Verbalia.

## Características

* Experiencia RPG en 3D.
* Historia original con progresión narrativa.
* Sistema de misiones y exploración.
* Desafíos educativos integrados en la jugabilidad.
* Interacción con NPCs y guardianes.
* Escenarios y personajes en 3D.
* Desarrollo basado en tecnologías web modernas.

## Tecnologías

* HTML5
* CSS3
* JavaScript (ES6+)
* Three.js
* Modelos GLTF / GLB

## Propósito Educativo

El proyecto busca fortalecer habilidades de ortografía y comprensión lectora mediante una experiencia interactiva donde el aprendizaje se integra de forma natural dentro de la aventura.
